# motivation-Maker-backend

이 프로젝트는 zenquotes.io API를 이용해 모티베이션 문구를 제공하고 즐겨찾기 기능을 지원하는 REST API 서버입니다

## Features
- JWT 기반 로그인, 회원가입 
- 모티베이션 문구 조회
- 즐겨찾기 추가 / 제거

## Roadmap
- 사용자별 즐겨찾기 관리
- 번역 기능(외국인 사용 가능하도록)

## Tech Stack
- Backend: NestJS, TypeORM
- Database: PostgreSQL
- Auth: JWT
- Language: TypeScript

## Related Repositories
- Frontend: https://github.com/aoyagi0105/motivation-Maker.git

## Installation
```bash
yarn install
```

## Run
```bash
yarn start:dev
```
